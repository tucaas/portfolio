Idea

This project focuses on the design of a complete website interface for a bicycle brand, exploring a strong visual language combined with a modular and scalable UI system.

The main goal was to create a bold and confident interface that highlights the products, communicates reliability, and guides users naturally through exploration, comparison, and contact.

Rather than starting from a formal briefing or wireframes, the project was developed directly through interface exploration and system definition inside a single Figma file.

Approach

The design process was UI-driven, focusing on visual hierarchy, layout consistency, and component reuse.

Each page was designed to work both independently and as part of a cohesive system, allowing the interface to scale across product pages, informational sections, and conversion flows.

Visual System

A strong visual system was defined to support the brand’s positioning, combining dark backgrounds, high-contrast typography, and accent colors to guide attention.

Typography choices prioritize clarity and impact, allowing product information and calls to action to stand out without competing with imagery.

Color usage was intentionally restrained, with accent colors applied primarily to interactive elements such as buttons, links, and highlights.

This approach reinforces visual hierarchy and improves scannability across dense product pages.

Components & Layouts

The interface was built using reusable components, allowing different page types to share the same visual language while serving distinct goals.

Product cards, feature lists, comparison tables, and call-to-action blocks were designed to be flexible and easily rearranged across layouts.

This modular approach supports scalability and consistency throughout the website.

Special attention was given to spacing, alignment, and rhythm, ensuring the interface feels structured and balanced even in content-heavy sections.

The result is an interface that remains visually clear while accommodating complex information.
